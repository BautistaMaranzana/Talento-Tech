Aviso de Accesibilidad

Nuestro compromiso con la accesibilidad

En Escape Literario, estamos comprometidos con nuestros suscriptores. Ya sea que estés utilizando tecnologías de asistencia como un lector de pantalla, una lupa, software de reconocimiento de voz o tecnología de interruptores, o si encuentras que tienes otras necesidades de accesibilidad, creemos que todos nuestros suscriptores deberían poder acceder y utilizar nuestro servicio a través de nuestro sitio web y sitio web móvil. Debido a que abrazamos los valores de la diversidad y la inclusión en todo lo que hacemos, nuestra misión es hacer de Scribd la opción para la lectura recreativa, para todas las personas.

¿Cómo logramos nuestros objetivos?

Establecemos altos estándares de accesibilidad web y nos esforzamos por garantizar que cumplimos con todas las leyes y pautas. Nuestro equipo de profesionales está dedicado a hacer que nuestras experiencias en el sitio web y el sitio web móvil sean lo mejor posible, y colaboramos con grupos de defensa, socios de la industria y proveedores para identificar áreas que necesitan mejoras.

  1. Las Pautas de Accesibilidad al Contenido Web (WCAG) 2.0 Nivel AA son reconocidas y aceptadas como la medida estándar internacional de éxito. Estamos trabajando para cumplir con esos estándares, y probaremos regularmente nuestro sitio para garantizar el cumplimiento. Estamos incorporando la accesibilidad y el diseño inclusivo como parte integral de todo nuestro trabajo digital: durante la planificación de proyectos, implementación y mejoras en el sitio basadas en la retroalimentación de los suscriptores.

  2. Estamos trabajando con grupos de defensa, especialistas en accesibilidad y usabilidad para asegurarnos de que nuestros sitios funcionen correctamente.

  3. Siempre estamos aprendiendo. Nuestros equipos aprenden de todo lo que hacemos, y siempre buscamos mejoras en el rendimiento, la velocidad y la accesibilidad.

Medidas que estamos tomando para mejorar la accesibilidad

Aquí hay algunos de los elementos que estamos abordando para asegurarnos de que nuestro sitio web y sitio web móvil sean lo más fáciles de usar posible para todos los suscriptores:

  1. Estructura del sitio: Estamos trabajando para utilizar encabezados apropiados, listas, párrafos y otras características de formato en scribd.com para que los suscriptores puedan entender el contenido y navegar más fácilmente con tecnología de asistencia.

  2. Equivalentes de texto: Las personas que no pueden ver imágenes o escuchar audio pueden necesitar alguna forma de equivalente de texto para interactuar con nuestros sitios. Estamos trabajando para asegurar que nuestras imágenes y multimedia sean completamente accesibles a través de texto alternativo, leyendas y transcripciones.

  3. Acceso completo con el teclado: Estamos trabajando para garantizar que todo lo que construimos pueda ser accedido utilizando un teclado.

  4. Consistencia del sitio: Estamos trabajando para lograr consistencia en la forma en que construimos nuestro sitio web y sitio web móvil. Una vez que entiendas cómo funcionan nuestra navegación y servicio, nuestro objetivo es asegurar que puedas interactuar con nuestro sitio web y sitio web móvil con funcionalidad consistente.
